# Variables

create a `<name>.tfvars` file with:

```
billing_account_id = "....."
```

or just provide it when running `plan/apply`

So:
1. `tf apply -var-file=<name>.tfvars`
2. `tf apply -var "billing_account_id=<my_billing_account_id>"`
